CemrgApp is as an MITK based framework extended with image processing and computer vision research software for:

  • Atrial Fibrosis Quantification ( documentation )
  • Cardiac Wall Strain Calculations
  • Cardiac Wall Thickness Calculations
  • Wireless Endocardial Pacing Optimisation
  • Cardiac Motion Measurement Tools

The source code can be accessed from: https://bitbucket.org/OrodRazeghi/cemrgapp/

The executable binaries can be downloaded from Github for:

  • MacOS (.dmg 177MB)
