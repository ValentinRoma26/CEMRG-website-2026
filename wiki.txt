This document works as an informal wiki to modify specific aspects of the website. 
There is no guarantee that this is the best way. Hopefully this document will be continuously 
updated.

### To view changes locally
Run in the terminal ``bundle exec jekyll serve --port 5000`` and open in a browser the address
[http://127.0.0.1:5000](http://127.0.0.1:5000)

### To view changes in the website

Commit and push the changes to gitlab. Close the jekyll server and open it again. Refresh the website.
Might take a bit because of the cache/cookies.

### Update collaborators

Add an entry to ``_data/collaborators.yml``.
